## Description

This component wraps `li`children of the ZCarousel.
It was conceived as a list of elements to respect the rules of accessibility. (https://www.w3.org/WAI/tutorials/carousels/working-example/)

## Usage

One `z-carousel`component wraps `N` `z-carousel-item`.
Any type of element can be passed to the `z-carousel-item`.
The size of the `z-carousel-item` fits to the size of the children

## How to use

Icon dimension is calculated using the link font size multiplied by 1.125. To change the font size set the --font-size-link variable in style; defaults to inherit (es. inline).

Current available text color is : primary, inverse, black and white. Primary and Inverse use the theming values from design-tokens; Black and White has fixed colors.


# ZLink primary
<Preview>
  <Story name="z-link primary">
    {html`
      <z-link
        href="${text('href', 'http://www.zanichelli.it')}"
        target="${text('target', '_blank')}"
        textcolor="${select("textcolor", ["primary", "inverse", "black", "white"], "primary")}"
        icon="${text('icon', 'download')}"
        iconposition="${select("iconposition", ["left", "right"], "left")}"
        htmlid="${number('htmlid', 1)}"
        htmltabindex="${number('htmltabindex', 1)}"
        isdisabled="${boolean('isdisabled', false)}"
        isactive="${boolean('isactive', false)}"
        underline="${boolean('underline', false)}"
      >z-link</z-link>`}
  </Story>
</Preview>

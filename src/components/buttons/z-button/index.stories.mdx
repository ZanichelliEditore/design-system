## Icon only

It is possible use a button with icon only passing an icon name to `icon` prop.
**Note:** `z-icon` drops its spacing accordingly.

| Prop name | Prop value        |
| --------- | ----------------- |
| icon      | Any `z-icon` name |

<Canvas>
  <Story
    name="Icon only"
    args={{
      variant: ButtonVariant.PRIMARY,
      size: ButtonSize.BIG,
      disabled: false,
      icon: "download",
    }}
    parameters={{
      controls: {
        exclude: ["square"],
      },
    }}
  >
    {(args) => html`
      <z-button
        variant="${args.variant}"
        size="${args.size}"
        disabled="${args.disabled}"
        icon="${args.icon}"
      ></z-button>
    `}
  </Story>
</Canvas>

## Zbutton as a link

It is possible for z-button to be an html `a` element.

To do this you need to set `href` prop.

<Canvas>
  <Story
    name="With href prop"
    args={{
      variant: ButtonVariant.PRIMARY,
      size: ButtonSize.BIG,
      disabled: false,
      icon: "arrow-quad-north-east",
      href: "https://wikipedia.com",
      target: "_blank",
    }}
  >
    {(args) => html`
      <z-button
        icon="${args.icon}"
        href="${args.href}"
        target="${args.target}"
        variant="${args.variant}"
        size="${args.size}"
        disabled="${args.disabled}"
        >Go to Wikipedia
      </z-button>
    `}
  </Story>
</Canvas>

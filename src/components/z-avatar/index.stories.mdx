## How to use

Pass at least one prop between **text** and **image**

If prop **image** contains a broken URL, the component will show **text** prop, if given

## N.B.

Using knobs is not possible to set a prop as optional. So when text field is empty the prop exists with empty value.
For example, when the background color field is empty no color is rendered, but when the component is used without this
prop is well rendered.

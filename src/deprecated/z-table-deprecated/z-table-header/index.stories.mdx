# ZTableHeader

## ⚠️ This component is deprecated and will be removed from the library on October 9th, please use the new `z-table` and the related components instead (`z-thead`, `z-tbody`, `z-tfoot`, `z-tr`, `z-th`, `z-td`).

### If you were using `Size` and `SortDirection` enums, use instead the string values directly (`"x-small"`, `"small"`, `"medium"`, `"large"`, `"special"`, `"asc"`, `"desc"`, `"none"`).

## Callback for sorting

Pass **onSort** prop as a function that accepts an object structured like this:

> { sortingOrientation: "asc" }

## Note

**columnId** prop is mandatory for sortable header otherwise there will be an error.

# ZTableDeprecated

## This component is deprecated and will be removed from the library on September 4th 2023, please use the new `z-table` and the related components instead (`z-thead`, `z-tbody`, `z-tfoot`, `z-tr`, `z-th`, `z-td`).

### If you were using `Size` and `SortDirection` enums (here or in the related components), use instead the string values directly (`"x-small"`, `"small"`, `"medium"`, `"large"`, `"special"`, `"asc"`, `"desc"`, `"none"`).
